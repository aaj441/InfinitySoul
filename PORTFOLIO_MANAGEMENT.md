```

  - Shadow manual processes
  - Interview key operators
  - Document workflow steps
  - Identify automation opportunities

Week 2-3: Design agent architecture
  - Map processes to agent roles
  - Define tool signatures and capabilities
  - Set up testing environment
  - Build initial agent prototypes

Week 4: Pilot deployment
  - Deploy agents to 10% of workload
  - Monitor quality and cost metrics
  - Gather feedback from team
  - Iterate on agent policies
```

```
  - Document new workflows
```

```
  - Set up monitoring dashboards
```

## Exit Strategy Framework

### When to Sell Checklist

```markdown
## Strategic Signals (Any 3 triggers exit planning)

- [ ] Valuation multiple reaches 10x+ EBITDA
- [ ] Strategic buyer expresses unsolicited interest
- [ ] Industry consolidation wave begins
- [ ] Regulatory changes favor large players
- [ ] VC funding dries up for competitors (buyer urgency)

## Operational Signals (Any 2 triggers exit)

- [ ] Maintenance cost > 30% of revenue
- [ ] Manual processes creeping back in
- [ ] Key team members want to leave
- [ ] Product roadmap misaligned with IS Fiber strategy
- [ ] Opportunity cost of management attention

## Market Signals (Any 2 triggers exit)

- [ ] Open-source alternatives gaining traction
- [ ] Compliance requirements weakening
- [ ] Market saturation (60%+ penetration)
- [ ] Commoditization (price compression)
- [ ] Customer acquisition cost rising faster than LTV
```

```typescript
    structure: 'cash' | 'equity' | 'earn_out'
  }

    arr: number
    margin: number
    valuation: number
  }
  
  // Operational

    agent_coverage: number
    cost_reduction: number
    quality_score: number

  // Exit
  exit_plan: {
    target_multiple: number
    target_date: Date
    potential_buyers: string[]
    status: 'not_ready' | 'preparing' | 'active'
  }
}

export class PortfolioRegistry {
  private products: Map<string, PortfolioProduct> = new Map()
  
  addProduct(product: PortfolioProduct): void {
    this.products.set(product.id, product)
  }
  
  getExitReady(): PortfolioProduct[] {
    return Array.from(this.products.values())
      .filter(p => p.metrics.valuation / p.metrics.arr > 10)
      .filter(p => p.automation.cost_reduction > 0.6)
  }
```
