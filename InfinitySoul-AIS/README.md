# Infinity Soul AIS v1.1

AI Insurance System-in-a-Box. Deploys in 5 minutes.

## Quick Start

1. `npm install`
2. Copy `.env.example` to `.env` and fill in keys
3. `npm run dev`
4. Open `http://localhost:3000`

## Modules

- A: AI System Scanner
- B: WCAG Accessibility Audit
- C: Data & Security Check
- D: Stress Test Engine
- E: NIST AI RMF Mapping
- F: Insurance Readiness Scoring
- G: Compliance Playbooks
- H: Evidence Vault
