# Infinity Soul AIS Manifesto v1.1

**We do not build AI that judges people.**
**We build AI that makes insurance fair.**

## Core Beliefs
1. Risk = Behavior, Not Identity
2. Data = Liability, Not Asset (Without Governance)
3. Compliance = Opportunity, Not Cost

## Tech Stack
- AI: Claude 3.5 (docs), GPT-4 (analysis), Kimi (research)
- Code: Next.js, Express, Supabase
- Infra: Vercel, Railway

## Build Rule
Every module must be **auditable in < 5 seconds** by a human.

## Revenue Rule
Every customer must **save 10x what they pay**.

## Deploy Rule
Ship daily. No exceptions.
