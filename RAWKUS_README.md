## Core Commands

### deploy_scout_agent

Deploy agent to find and acquire distressed MGAs.

```bash
./bin/rawkus deploy_scout_agent \
  --target "mga" \
  --criteria "cr>115,premium<20m,age>55" \
  --offer "$50K + 15% revshare" \
  --count 3 \
  --execute
```

**Options:**
- `--target` - Target type (default: "mga")
- `--criteria` - Filter criteria (default: "cr>115,premium<20m,age>55")
- `--offer` - Offer structure (default: "$50K cash + 15% revshare + 1,000 $SOUL")
- `--count` - Number of targets to contact (default: 3)
- `--execute` - Execute immediately (omit for dry run)

**Expected Output:**
- 33% response rate within 24 hours
- $12M premium acquired per successful deal
- 240x ROI in Year 1

### deploy_underwriting_agent

Replace human underwriters with AI automation.

```bash
./bin/rawkus deploy_underwriting_agent \
  --cell "mga_001" \
  --data_path "claims_csv"
```

**Options:**
- `--cell` - MGA cell identifier
- `--data_path` - Path to claims data

**Expected Output:**
- 30-second quote generation (vs 3 days human)
- $0.10 per quote (vs $100 human)
- 99.9% cost reduction

### deploy_governance_agent

Automate governance decisions through House Committee voting.

```bash
./bin/rawkus deploy_governance_agent \

  --vote
```

**Options:**

**Expected Output:**

### fire_underwriters

Reduce costs through automation.

```bash
./bin/rawkus fire_underwriters \
  --count 10 \
  --confirm
```

**Options:**

- `--confirm` - Confirm the action

**Expected Output:**

- 10 positions = $1M annual savings
- Debt neutralized in 4.8 months

### harden_protocol

Secure protocol from intrusions.

**Options:**

**Expected Output:**

### status

Show debt neutralization progress.


**Options:**
