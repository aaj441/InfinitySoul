## üöÄ Quick Start (5 Minutes)

- Node.js 20+

### Installation

```bash

# 2. Install dependencies

# 5. Start the application
```

## üó≥Ô∏è Governance

Infinity Soul is governed by its users through **on-chain voting** (coming soon: Ethereum L2).

### How It Works

1. **Propose**: Any user can create a proposal (e.g., "Add Garmin integration")
2. **Vote**: 7-day voting period (yes/no/abstain)
3. **Execute**: If passed (50% + 1), the team implements it within 30 days

## üôã FAQ

**Q: Is my biometric data secure?**

**Q: How do you make money?**

**Q: Can I build custom agents?**
